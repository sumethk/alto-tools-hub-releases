# Alto Tools Hub Downloads

Official Windows downloads and automatic updates for Alto Tools Hub.

## Download

Download the latest version from [Releases](https://github.com/sumethk/alto-tools-hub-releases/releases/latest).

- `AltoToolsHub-vX.Y.Z-Windows.zip` contains the MSI and guided installation helper.
- `AltoToolsHubSetup.msi` is the Windows per-user installer.
- `AltoToolsHubUpdate.zip` is used by existing installations for in-app updates.
- Matching `.sha256` files can be used to verify download integrity.

This public repository contains distribution files only. The application source code,
build system, and development history are maintained in a separate private repository.
